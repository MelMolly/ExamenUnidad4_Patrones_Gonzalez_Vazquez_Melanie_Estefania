# ExamenUnidad4_Patrones_Gonzalez_Vazquez_Melanie_Estefania

Este examen está basado en el examen de la unidad 3, un gestor de audio con efectos sonoros opcionesl, En esta unidad se emplea 4 patrones de diseño.

Decorador: Se usó para crear los efectos y agregarlos al audio base

Factory: Se usó para crear los objetos de cada efecto para mandarlos a llamar en la consola

State: Se usó para declarar los estados de play, stop y pausa de la música

MCV: Se usó para la estructura del programa

Se deberá colocar en la consola una ruta de archivo .mp3 o .wav de audio: ejemplo C:\Users\dell\Music\luna.mp3
Lo mandará a un menú donde puede escoger en reproducir, pausar, detener, agregar efecto, restaurar audio original y salir. Cada que se quiera agregar un efecto, deberá detener la música primero.
